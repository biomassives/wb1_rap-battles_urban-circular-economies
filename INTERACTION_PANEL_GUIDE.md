## How to Use

### Getting Started

1. **Select Your Role**
   - Click on role card that matches your interests
   - Role description appears below
   - Activity sections filter automatically

2. **Explore Activities**
   - Browse cards in your relevant sections
   - View open opportunities
   - Check your active projects

3. **Take Action**
   - Click "+ New Battle" / "+ Start Collab" etc.
   - Join existing activities
   - Contribute to projects

4. **Track Progress**
   - Monitor your active items
   - View completion status
   - Earn XP for participation

### Quick Actions

Each section has action buttons:

**Rap Battles**:
- "+ New Battle" - Create challenge
- "Accept" - Join open battle
- "Judge" - Vote on battle

**Reggae Collaborations**:
- "+ Start Collab" - Begin new project
- "Join" - Contribute to collab
- "Connect" - Match with artist

**Environmental Tech**:
- "+ New Project" - Create project
- "Contribute" - Add to project
- "Help" - Take on task

**Food Security**:
- "+ Start Garden" - Create garden
- "Join" - Become participant
- "Request/Offer" - Exchange items

### Earning XP

Activities award XP:
- **Role Exploration**: +5 XP per role viewed
- **Creating Battle**: +50 XP
- **Winning Battle**: +150 XP
- **Judging Battle**: +10 XP per vote
- **Starting Collab**: +30 XP
- **Joining Collab**: +20 XP
- **Code Contribution**: +25 XP per commit
- **Joining Garden**: +40 XP
- **Workshop Completion**: +60 XP
