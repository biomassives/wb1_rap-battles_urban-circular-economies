# Airdrop Rewards System Guide

## Overview

The **Airdrop Rewards System** is a scheduled token distribution mechanism that incentivizes user participation across **artist promotion**, **educational projects**, **mentorship**, and **events**. Users earn tokens and XP by completing actions synchronized with platform activities.

## Access

Visit: `http://localhost:4321/airdrops`

## System Components

### 1. **Token Balance**
- Track your earned tokens

- View claim history

### 2. **Active Airdrops**
Currently claimable rewards based on completed actions.

### 3. **Upcoming Airdrops**
Scheduled future rewards to plan participation.

### 4. **25 Reward Templates**
All possible reward types organized by category.

## Airdrop Scheduling

### How Airdrops Cycle

Rewards rotate through scheduled drops:

1. **Weekly Drops** (Every Monday)
   - Stream Boost
   - Constructive Feedback
   - Battle Spectator
   - Research Contributor
   - Mentor Session

2. **Bi-weekly Drops** (1st & 15th)
   - Share Campaign
   - Quiz Master
   - Peer Group Session

3. **Monthly Drops** (1st of month)
   - Artist Discovery
   - Collab Support
   - Playlist Creator
   - Course Completion
   - Project Submission
   - Peer Teacher
   - Mentorship Milestone
   - Community Day

4. **Event-Based Drops**
   - Live Event Attendance (when scheduled)
   - Battle Event (tournament dates)
   - Hackathon (event dates)
   - Milestone Celebration (milestone dates)

5. **One-time/Certification Drops**
   - Certification Earned (upon completion)
   - Become a Mentor (upon qualification)
   - Success Story (quarterly)

**Q: Can't claim reward?**
- Ensure requirements are met
- Check if already claimed
- Verify wallet connected
- Try refreshing page

**Q: Balance not updating?**
- Check localStorage
- Verify wallet address
- Clear cache and retry
- Check browser console

- Check date/time
- Verify you're in schedule window
- Refresh airdrops page
- Check category filter

- Tokens stored in localStorage
- Tied to wallet address
