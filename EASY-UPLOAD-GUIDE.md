## ğŸ”§ Troubleshooting

### "File won't upload"

**Check:**
- File is audio format (WAV/MP3/OGG)
- File size is under 5 MB
- Browser storage isn't full

**Try:**
- Refresh the page
- Use smaller file
- Clear browser cache

### "Can't find upload button"

**Look for:**
- Small blue **+** button
- Top-right corner of each pad
- Appears on all 18 pads

**On mobile:**
- Button is slightly larger (easier to tap)
- May need to zoom in slightly

### "Drag and drop not working"

**This is normal on mobile!**
- Mobile doesn't support drag-and-drop
- Use the **+ button** instead
- Works perfectly on phones

**On desktop:**
- Make sure file is audio format
- Try clicking + button instead
